# TekMonks Assignment - Time.com Latest Stories API

This is a simple Node.js project made for the TekMonks internship assignment.
The task was to fetch the latest 6 news stories from the Time.com homepage
and return them in JSON format. No external libraries or npm packages were allowed.

## How to Run

1. Make sure Node.js is installed
2. Open the project folder in Command Prompt / Terminal
3. Run the command:
   node server.js

## API Endpoint

Open this link in the browser:
http://localhost:8000/getTimeStories

This will return the latest 6 stories in JSON format.

## Example Output

[
  {
    "title": "Sample story title",
    "link": "https://time.com/sample"
  }
]

## Rules Followed

- No external libraries used
- Only built-in modules: http and https
- No regex used
- Basic code, written manually without AI
